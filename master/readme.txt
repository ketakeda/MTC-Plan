
マスターデータコンバーター



■ 環境設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

マスターデータの元のファイルは DropBoxに保存されているので
保存先フォルダーを得るために環境変数を設定してください。

システムプロパティを開いて環境変数の設定を行ってください。

ユーザー環境変数に以下の設定をしてください。
変数  MTC-Developer
値    E:\Dropbox (ガンホー・オンライン・エンターテイメント)\MTC-Developer
      ※こちらは実際の使用するPCの環境に合わせてください。


■ 定義ファイル
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



□ ヘッダーを定義している行の設定
───────────────────────────────────

ラベル     : HeaderDefRow
パラメータ : 行番号

例)
HeaderDefRow
	2

□ 出力フォーマット
───────────────────────────────────

ラベル     : Format
パラメータ : ヘッダー出力フォーマット

例)
Format
	ヘッダー名, カラム名, 型
